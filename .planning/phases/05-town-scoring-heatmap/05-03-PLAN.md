## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-09 | DoS | division by zero on a degenerate `max==min` range in normalize | mitigate | `normalize` throws when `hi <= lo` (validate-and-throw); Task 1 asserts the throw |
| T-05-10 | Tampering | NaN/Infinity poisoning the composite (would later make canonicalJson throw) | mitigate | all math in the `Dec` clone; clamp to [0,1]; missing → null (never 0/NaN); Task 2 asserts null edges |
| T-05-11 | Tampering | float re-entry via `Number()` on dollar/score values | mitigate | composite stays a `Dec` decimal string; bucket compares `Money.toCents()` bigints; source asserts no `Number(` on money/score |
| T-05-12 | Tampering | missing metric imputed as 0 (biases the town to worst — D-03 violation) | mitigate | drop + renormalize present weights; composite null when nothing present; Task 2 asserts never `'0'` |
</threat_model>
