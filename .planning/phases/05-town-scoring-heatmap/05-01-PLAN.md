| T-05-01 | Tampering | float/non-canonical metric value smuggled into a row | mitigate | every metric `value` is `decStr` (canonical-decimal regex) — a float fails parse; asserted in Task 3 |
| T-05-02 | Tampering | unknown/extra key on a row | mitigate | `townRowSchema.strict()` rejects unknown keys; asserted in Task 3 |
| T-05-03 | Tampering | missing metric silently imputed as 0 (data poisoning the future score) | mitigate | D-03 enforced at the data layer — missing = absent key; Task 3 asserts `undefined`, never 0 |
